### rust-radio-in-100-lines
`command-line streaming internet radio in 100 lines of Rust code.`

![image](https://github.com/user-attachments/assets/d0525ff4-59a3-4527-a658-54ca14ba5836)

`cargo run --release` and enjoy the jams. Thank you to the [radiobrowser](https://www.radio-browser.info/) project for the radio stations I used during testing. This was a learning project done in preparation for a larger project, and I learned a lot about using `tokio` and managing audio streaming, buffering, decoding in my quest for smooth playback.
